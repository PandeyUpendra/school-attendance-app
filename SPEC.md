## 7. APP ENTRY POINT — `lib/main.dart`

```
main()
  → Firebase.initializeApp()
  → FirebaseAuth.signInAnonymously()  ← NEW (for Storage/Firestore rules)
  → runApp(SchoolApp)
       → MaterialApp(home: _SplashGate)
            → _SplashGate._checkSession()
                 → AuthService.getSession()
                 → BaseFirestoreService.currentSchoolId = session['schoolId']  ← NEW
                 │
                 ├── role='teacher'    → TimetableService.getTeacherById(schoolId, id) → HomeScreen(teacher)
                 ├── role='coordinator' → CoordinatorDashboard
                 ├── role='principal'   → PrincipalDashboard
                 ├── role='guardian'    → (1 child) GuardianDashboard(schoolId, class, roll)
                 │                      → (N children) StudentSelectionScreen(schoolId, links)  ← NEW
                 └── (none)            → RoleSelectionScreen
```
